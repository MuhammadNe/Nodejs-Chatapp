# Nodejs-Chatapp
An example of a Nodejs Chatapp using Redis Pub/Sub

### Clone the project: 
`git clone https://github.com/MuhammadNe/Nodejs-Chatapp.git`
### Then run: 
`cd ./Nodejs-Chatapp && npm install`

## Available Commands
### `npm start`
Runs the app in production mode.

### `npm run dev`
Runs the app in dev nodemon mode.
